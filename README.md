# SyntaxWear - Tênis e Sneakers Online

SyntaxWear é um projeto de e-commerce de página única para uma loja de calçados online. A página foi desenvolvida com HTML5 e CSS3, com foco em um layout responsivo e moderno.

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

```
/
├── css/
│   ├── base.css
│   ├── reset.css
│   ├── variables.css
│   └── components/
│       ├── footer.css
│       ├── header.css
│       ├── hero.css
│       ├── product-category.css
│       └── product-grid.css
├── images/
│   ├── banners/
│   ├── favicons/
│   ├── icons/
│   ├── logo/
│   └── products/
├── index.html
└── README.md
```

- **`css/`**: Contém os arquivos de estilo.
  - **`base.css`**: Estilos básicos para o corpo, tipografia, etc.
  - **`reset.css`**: Reseta os estilos padrão do navegador.
  - **`variables.css`**: Define as variáveis de cores e outras propriedades.
  - **`components/`**: Contém os estilos para os componentes da página.
- **`images/`**: Contém todas as imagens do projeto, organizadas por tipo.
- **`index.html`**: A página principal do e-commerce.
- **`README.md`**: Este arquivo.

## Tecnologias Utilizadas

- **HTML5**: Para a estrutura da página.
- **CSS3**: Para a estilização e layout.

## Páginas e Componentes

A página principal (`index.html`) é dividida nas seguintes seções:

- **Header**: O cabeçalho da página, com o logo, menu de navegação e ícones de acesso rápido.
- **Hero Section**: A seção principal da página, com uma imagem de destaque e um call-to-action.
- **Categories Section**: Uma seção com as categorias de produtos.
- **Product Grid**: Uma grade com os produtos em destaque.
- **Footer**: O rodapé da página, com informações de contato, links de navegação e redes sociais.

Os estilos CSS são modularizados em componentes, facilitando a manutenção e o reuso de código.
